# LookBag 查询背包

- 作者: 羽学
- 出处: 无
- 这是一个Tshock6.1.0服务器插件适用于Terraria1.4.5主要用于：使用/bag指令查询玩家背包，可将对方背包复制到自身后,再设置给目标玩家,支持离线查询与设置。

## 指令

| 语法           |        权限         |   说明   |
| -------------- | :-----------------: | :------: |
| /bag | bag.use | 菜单指令(恢复自己背包) |
| /bag 玩家名 或 索引 | bag.use | 查询指定玩家背包 |
| /bag g 玩家名 或 索引 | bag.admin | 打开玩家背包（并备份自己） |
| /bag s 玩家名 或 索引 | bag.admin | 修改玩家背包（支持离线/死亡延迟） |

## 使用方法

- 使用/bag g 玩家名 的背包数据复制到自身
- 修改物品后使用/bag s 玩家名 设置给对方
- 使用/bag 菜单指令还原自身背包
